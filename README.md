# Résumé du répertoire:

Ce répertoire contient un rapport analytique portant sur la vaccination contre la COVID-19. Celui a été créé sur RStudio grâce au packages [Distill](https://rstudio.github.io/distill/).

## Packages requis:

- tidyverse
- lubridate
- readxl
- magick
- webshot
- kableExtra
- hrbrthemes
- zoo
- countrycode
- ggtext
- moderndive
- broom
- data.table
- maps
